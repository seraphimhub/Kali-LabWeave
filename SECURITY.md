# Security Policy

Kali LabWeave adalah alat dokumentasi lab dan tidak dirancang untuk menjalankan aktivitas ofensif.

## Melaporkan Masalah

Jika menemukan bug keamanan pada tool ini, buka issue privat atau hubungi maintainer repository. Sertakan:

- versi tool,
- sistem operasi,
- langkah reproduksi,
- dampak yang terlihat,
- file contoh bila aman dibagikan.

Jangan lampirkan rahasia, token, credential, atau data pihak ketiga.
